London

London is one of my favourite places to go in my holidays. Only 2 hour flight from Helisnki airport and you are in Heathrow.

It is pretty easiy to get anywhere from Heathrow by using the underground or "tube" as the locals call it.

Few places where you defintely want to go in your trip to london:

  1. London Eye
  2. Big Ben
  3. Buckinham Palace
  4. Trafalgard Square
  5. Picadilly Circus
